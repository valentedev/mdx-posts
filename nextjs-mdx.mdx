## What is MDX?

- Create a github repository with all your posts;
- Use MDX Remote (compileMDX) to get meta fields from Post files in from Github repo with `getPostByName()`
- Pull the list of file from Github and generate an array of files with `getPostsMeta()`

Both fields must match

```jsx showLineNumbers
const { frontmatter, content } = await compileMDX<{
    // These files must match github blog posts
    title: string;
    author: string;
    date: string;
    excerpt: string;
    categories: string[];
  }>({
    source: rawMDX,
    options: {
      parseFrontmatter: true,
      mdxOptions: {
        rehypePlugins: [
          rehypeHighlight,
          rehypeSlug,
          [
            rehypeAutolinkHeadings,
            {
              behavior: "wrap",
            },
          ],
        ],
      },
    },
  });
```
